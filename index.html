Project Description

This is a data entry form for Heroes of The Storm matches in order to better track KDA ratios among amateur players. Data is stored in a MongoDB.
Built from scratch, not using templates or reusing existing projects. Originally created using all JQuery-created DOM elements. Later, I wanted to recreate it as a node.js app with pug and HTML equivalents, replacing the JQuery. All logic-checking code will be added soon.
